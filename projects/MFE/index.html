MFE(Machine Fingerprint Encryption) is a custom encryption extension written in C#. I used this custom built tool to encrypt locally stored data.

The machine fingerprint will be used in extension with other known encryption methods to further encrypt the string before it will be used/stored

Note that the process of encryption is still a thought, and until the extension is done it surely will change.
